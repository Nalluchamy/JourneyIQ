# JourneyIQ Documentation Portal

Welcome to the official documentation portal for **JourneyIQ**—a behaviorally intelligent SaaS platform designed for modern retail storefronts.

## Core Documentation Map

- **[System Architecture](architecture.md)**: Deep dive into systems design, recommendation engine pipeline flows, and database schemas.
- **[Production Setup](PRODUCTION_SETUP.md)**: Complete step-by-step local and production installation guide.
- **[SaaS Deployment Guide](DEPLOYMENT.md)**: Details cloud deployments on Render, Railway, AWS, Azure, and static frontend hosts.
- **[DevOps Operations](DEVOPS.md)**: Covers Docker Compose orchestration, NGINX proxy configs, logging architecture, and CI/CD pipelines.
- **[MLOps Pipeline](MLOPS.md)**: Explains PyTorch NCF model training, versioned registry controls, inference metrics tracking, and rollbacks.
- **[Database & Caching Guide](DATABASE_GUIDE.md)**: Hardened Supabase schema design, query indexes, and Redis caching.
- **[Disaster Recovery & Backup](BACKUP.md)**: Step-by-step procedures for database backups, checksum verification, and restoring data.
- **[Monitoring Runbook](RUNBOOK.md)**: Curl commands, service recovery routines, common issues, and Prometheus configurations.
- **[Production Checklist](PRODUCTION_CHECKLIST.md)**: Final deployment verification checklist for stable Release Candidate (v1.0.0).
